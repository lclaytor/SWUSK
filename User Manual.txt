Feedback Survey User Manual

OPERATION
  To take a survey, the user should select a response to each question that appears by clicking one of the buttons in the middle of the screen. After selecting an answer, clicking "Next" will save the answer and retreive the next question. Once all of the rating questions have been answered, the user may leave any additional comments by typing in the text box that appears. Once complete, clicking "Submit" will save and send all of the answers selected.
